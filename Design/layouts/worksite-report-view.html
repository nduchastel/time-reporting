Worksite Report Features:

  • Multiple Views: By Worksite (default), By Worker, By Date
  • Flexible Filtering: Date range, specific worksite, specific worker
  • Summary Statistics: Total hours, active workers, active sites, averages
  • Daily Breakdown: Hours per day per worker at each site
  • Site Totals: Aggregate hours per site for billing/job costing
  • Worker Totals: Aggregate hours per worker for payroll
  • Visual Distribution: Bar charts showing how workers' time is distributed across sites
  • Peak Insights: Busiest day, most active worker per site
  • Export Options: CSV (for Excel/payroll), PDF (for billing/records)
  • Drill-Down: Click "Details" to see individual time cards, "View Log" to see all log events for that site

Use Cases:

💰 Client Billing
Export site-specific hours for invoicing clients. "Simons Property: 82.5 hrs @ $X/hr"
💵 Payroll Processing
Export worker totals for payroll. CSV format compatible with payroll systems.
📊 Job Costing
Analyze labor costs per site. Which sites are most labor-intensive?
📈 Resource Planning
Identify peak days, busy sites, and worker utilization patterns for scheduling.
