Directed Voice Sample System:

  • Standardized Phrases: Manager configures exact text all workers will read (not free-form)
  • Ground Truth: We know exactly what text was spoken, so we can send it as Whisper context
  • Sample 1 Auto-filled: Worker's name is filled automatically from their profile
  • Samples 2-5 Customizable: Manager picks relevant worksites and activities for their company
  • Clear Instructions: Workers see "READ THIS PHRASE:" with the exact text to say
  • Context Accumulation: All workers' sample texts become vocabulary hints for the transcription system
  • Per-Language: Each language (EN/FR/ES) has its own configured phrases

Data Flow:

1. Manager configures phrases:
  Sample 2: "Simons Property on Main Street"
  Sample 3: "I worked on road repair and snow removal"

2. Bob reads those exact phrases during onboarding

3. Database stores:
  { audio: blob, text: "Simons Property on Main Street" }
  { audio: blob, text: "I worked on road repair and snow removal" }

4. Bob records time entry: "I did 8 hours at Simons today"

5. System sends to Whisper with context:
  prompt: "Bob Martinez, Simons Property on Main Street, road repair, snow removal"

6. Whisper recognizes "Simons" correctly (primed by context)
